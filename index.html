Welcome to GYPC

Gypc is an easy-to-use website that allows you to get in a few clicks a personalized cover for your playlist. The only thing you need is 10 seconds to choose the mood and the theme that your playlist is about. This is a beta version of the website, it will be upgrated in the nexts months. It will get bigger and bigger, so stay tuned for more functions!

How it works

GYPC's mechanism is very simple: for each mood (dropdown bottom) there are 9 theme that users can choose. Each decision give differents results: mood for the color (from psychological studies) and theme for the subject.
